# Screen Docs Index

Tài liệu theo màn hình để AI/dev hiểu nhanh mục tiêu UI, flow và state.

## Danh sách màn hình

- `screen-map.md` - bảng tra cứu nhanh route/component/state/i18n
- `screen-map-graph.md` - flowchart Mermaid của toàn bộ luồng màn hình
- `routing-tech-graph.md` - flow kỹ thuật redirect/rewrite locale + subdomain
- `login.md` - màn hình đăng nhập
- `register.md` - màn hình đăng ký
- `reset-password.md` - màn hình quên mật khẩu
- `select-platform.md` - màn hình chọn nền tảng
- `platform-dashboard.md` - dashboard theo platform + chọn bot
- `not-found.md` - màn hình 404

## Quy tắc đọc nhanh

1. Xác định màn hình cần sửa.
2. Đọc mục `Input/Output flow`.
3. Đọc mục `State` trước khi thay đổi logic.
4. Đối chiếu i18n key trước khi thêm text.
