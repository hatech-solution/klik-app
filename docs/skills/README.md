# Project Skills

Thư mục này mô tả skill dùng cho AI agent trong repo.

## Skill hiện có

- `skills/klik-onboarding/SKILL.md`

## Lưu ý

- Skill được đặt trong `skills/` để mọi AI agent đọc được từ repo, không phụ thuộc `.cursor`.

## Mục đích

Skill này giúp agent:

- onboard nhanh dự án,
- tránh phá flow subdomain/i18n,
- bám đúng checklist verify (`yarn lint`, `yarn build`, test flow chính).
